# 在windows上查看端口占用的进程

根据端口号查看进程号
```
netstat -aon | findstr 8000
```

![](./端口占用查看/15719736518763.png)
{% asset_img 15719736518763.png %}

根据进程号查看进程
```
tasklist | findstr 2908
```

![](./端口占用查看/157197368922.png)
